DNS система доменных имен - это технология интернет среды, система, которая хранит и обрабатывает информацию о доменных зонах.
DNS необходима для соотнесения IP адресов и имен сайтов.
Имеет иерархию, где каждый узел имеет свое уникальное имя. Точка «.» — это самый верхний уровень иерархии ДНС, а уже от него идут поддоменные зоны .ru, .com и т.д.

В чём разница протоколов HTTP/HTTPS и их версий
HTTP – это протокол передачи гипертекста (ссылка). Способ передачи любой информации в сети.
По умолчанию используется 80й порт
HTTPS – это защищенный протокол передачи гипертекста. Продвинутая и безопасная версия HTTP
Позволяет создавать защищенное и зашифрованное соединение между браузером и сервером. Использует 443 порт.
Различия:
HTTP передает данные открытым текстом, HTTPS – зашифрованным
HTTP Работает быстрее за счет того, что не нужно шифровать канал связи
Разные порты для работы 80 у HTTP и 443 для HTTPS
HTTP 1.1 – версия протокола, которая используется сейчас
HTTPS обладает версией TLS 1.0, 1.1, 1.2 (SSL более не используется, т.к не может обеспечить должного шифрования)
